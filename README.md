# Flutter Counter — Cupertino Design

App Flutter con counter incrementabile e decrementabile.

## Caratteristiche
- Design **Cupertino** (iOS-style)
- Primary color: **Blue**
- Bottoni + e - con feedback haptic-style
- Counter con animazione
- Pulsante reset
